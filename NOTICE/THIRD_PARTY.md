# Third-Party Notice

tooling such as Python, Next.js, React, Tailwind CSS, ESLint, and optional
design or agent skills.

Project-local skill packs are not included in generated starter projects by
default. If you install or copy third-party skills into a generated project,
preserve their original licenses and notices.

Generated SQLite files under `data/` are derived local state and are not part
of the licensed source distribution.
